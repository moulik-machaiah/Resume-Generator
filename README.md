# 📄 Automated Resume Builder

A modern web-based resume builder that allows users to create, edit, and export professional resumes through an intuitive multi-section interface. The application supports dynamic form inputs, structured data handling, and PDF generation with consistent formatting.

---

## 🚀 Features

- 🧩 Modular Multi-Section Forms  
  Input personal details, education, skills, projects, internships, and more.

- 🔄 Dynamic Field Management  
  Add/remove sections like projects, skills, and internships seamlessly.

- 🧠 Centralized State Management  
  Efficient handling of all form data for smooth navigation and editing.

- 📄 PDF Resume Generation  
  Generates well-structured resumes with proper formatting using jsPDF.

- 💾 JSON Export & Import  
  Save resume data and reload it later for editing.

- 📸 Profile Photo Upload  
  Add and preview a profile image in the resume.

---

## 🛠️ Tech Stack

## ⚙️ How to Run

1. Clone the repository:
   git clone <your-repo-link>
   cd resume-builder

2. Open the project:
   - Double click `resume_builder_ui.html`
   OR
   - Open it in any web browser

---

## 📌 Usage

1. Select **Create Resume**
2. Fill in all sections:
   - Personal Info
   - Summary
   - Education
   - Skills
   - Projects
   - Internships, etc.
3. Navigate using the sidebar
4. Click **Generate PDF**
5. Download your resume
